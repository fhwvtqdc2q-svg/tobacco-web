# Localization Policy / سياسة الترجمة ودعم اللغة

**Effective Date / تاريخ النفاذ:** `2026-06-30`

This document describes how **TOBACCO Web** approaches localization, multilingual support, and Arabic language handling.

---

## English

### 1. Supported languages
The service may support one or more interface, content, translation, or input languages depending on the product version and integrations in use.

### 2. Arabic language support
Where Arabic is supported, we aim to provide a reasonable user experience for:
- Arabic text input,
- Arabic text display,
- UTF-8 encoded content,
- mixed Arabic/Latin content,
- right-to-left (RTL) layout considerations where applicable.

### 3. Limitations
Arabic support may vary depending on:
- operating system,
- browser,
- font availability,
- document rendering engines,
- third-party APIs,
- OCR quality,
- file format constraints.

We do not guarantee perfect rendering, translation quality, punctuation handling, bidirectional text behavior, or contextual correctness in every case.

### 4. User responsibilities
Users should review translated, extracted, or rendered Arabic content before publishing, legal use, or operational dependency.

### 5. Reporting problems
Language display, translation, shaping, or RTL issues can be reported to: **ozk.kh@outlook.com**

---

## العربية

### 1. اللغات المدعومة
قد تدعم الخدمة لغة واحدة أو أكثر على مستوى الواجهة أو المحتوى أو الترجمة أو الإدخال، وذلك بحسب إصدار المنتج والتكاملات المستخدمة.

### 2. دعم اللغة العربية
عند توفر دعم العربية، فإننا نسعى لتقديم تجربة مناسبة فيما يتعلق بـ:
- إدخال النص العربي،
- عرض النص العربي،
- المحتوى المشفر بصيغة UTF-8،
- المحتوى المختلط بين العربية واللاتينية،
- مراعاة اتجاه الكتابة من اليمين إلى اليسار (RTL) عند الاقتضاء.

### 3. القيود
قد يختلف مستوى دعم العربية بحسب:
- نظام التشغيل،
- المتصفح،
- توفر الخطوط،
- محركات عرض المستندات،
- واجهات API الخارجية،
- جودة OCR،
- قيود تنسيقات الملفات.

ولا نضمن التمثيل المثالي أو جودة الترجمة أو معالجة علامات الترقيم أو سلوك النص ثنائي الاتجاه أو الدقة السياقية في جميع الحالات.

### 4. مسؤولية المستخدم
ينبغي على المستخدم مراجعة المحتوى العربي المترجم أو المستخرج أو المعروض قبل نشره أو استخدامه قانونيًا أو الاعتماد عليه تشغيليًا.

### 5. الإبلاغ عن المشكلات
يمكن الإبلاغ عن مشكلات عرض اللغة أو الترجمة أو تشكيل الحروف أو RTL عبر: **ozk.kh@outlook.com**
