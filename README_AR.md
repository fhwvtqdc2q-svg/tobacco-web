# TOBACCO Web

منصة خدمة عملاء عربية تعمل من المتصفح على Windows وiPhone وMac، ومتصلة بـ Supabase لحفظ الطلبات.

الرابط العام الحالي:

```text
https://fhwvtqdc2q-svg.github.io/
```

## الحالة الحالية

- تسجيل الدخول يعمل عبر Supabase.
- جدول الطلبات `customer_requests` يعمل.
- الطلبات تحفظ في قاعدة البيانات.
- يمكن تصدير الطلبات بصيغة CSV لفتحها في Excel وتجهيزها للتوافق مع برنامج الأمين.
- توجد صفحة `الأمين` لرفع جرد الأمين ولائحة الأسعار اليومية.
- صفحة `الأمين` تعرض تنبيهات قرب النفاد، المواد الراكدة، المواد الفعالة، والمقارنة مع التقرير السابق.
- يمكن تنزيل لائحة أسعار جديدة لا تحتوي إلا المواد الموجودة في المستودع والتي لديها سعر صالح.

## التوافق مع Mac وiPhone

هذا المشروع يعمل كتطبيق ويب PWA على Safari. إذا أردنا لاحقا تطبيق iPhone أصلي على App Store، سنحتاج Xcode على Mac أو GitHub Actions macOS runner للبناء والتوقيع.

## التوافق مع الأمين للمحاسبة

الخطوة الحالية هي الربط عبر ملفات Excel/CSV المصدرة من الأمين. من صفحة `الأمين`:

1. ارفع ملف الجرد التجميعي.
2. ارفع ملف لائحة الأسعار اليومية.
3. اختر حد تنبيه قرب النفاد.
4. اضغط `تحليل وحفظ التقرير`.
5. اضغط `تنزيل أسعار المتوفر فقط` للحصول على لائحة أسعار لا تحتوي المواد غير الموجودة أو المواد ذات السعر الصفري.

عند تسجيل الدخول وحفظ التقرير في Supabase، يظهر آخر ملخص على iPhone عند فتح الرابط العام.

## الملفات المهمة

- `src/app.js`: الواجهة والمنطق.
- `src/supabase-client.js`: طبقة Supabase.
- `src/config.js`: إعدادات Supabase العامة.
- `supabase/schema.sql`: جداول الطلبات وتقارير الجرد وسياسات RLS.
- `supabase/permissions-fix.sql`: إصلاح صلاحيات الطلبات وتقارير الجرد عند الحاجة.
- `.github/workflows/pages.yml`: نشر GitHub Pages.
